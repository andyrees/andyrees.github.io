Programming to me is the ability to elevate the mundane into the elegant. Take a repetitive task and spend time coding a solution and you not save any time. But try that a thousand times and there is no contest.

As many others have, I'm sure I started with BASIC in the 70s on a BBC Micro, Assembly code at University, then C and C++ later, Java & C#. Python allowed me to enjoy programming again; once you get over the white space issue. But my recent obsession is Golang.

Python might be nice to write, but it can be messy and complex to deploy. However, Go has inferred runtime static typing, compiles to an executable with no runtime dependencies and is garbage collected and is as much a pleasure to code as python.

Go is increasingly the answer for both small and large projects, in my day to day activities. My recent adventures with broadcast regulations and perceived loudness, ended with quite a nice example of Go's effectiveness resulting in a really useful utility.
